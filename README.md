# Remi Sharoon — Data Engineering Portfolio

This repository contains the source code for [remisharoon.com](https://cars-analyzer-react.pages.dev/) — a modern React + Bootstrap
website that showcases Remi’s experience, skills, and flagship data products.

![Hero section preview](public/favicon.svg)

## ✨ Features

- **Hero & positioning** that capture Remi’s data engineering value proposition and core strengths.
- **Experience timeline** highlighting impact across leadership and hands-on roles.
- **Skills, certifications, and metrics** to quickly communicate technical range and credibility.
- **Project spotlights** with direct access to the live Cars Analyzer dashboard hosted on Cloudflare Pages.
- **Writing & contact sections** linking to Medium, LinkedIn, and résumé resources.

## 🛠️ Tech stack

- [React 18](https://react.dev/)
- [Vite 5](https://vitejs.dev/)
- [Bootstrap 5](https://getbootstrap.com/)

## 🚀 Local development

```bash
npm install
npm run dev
```

Vite will print a local development URL (default: `http://localhost:5173`).

## 📦 Production build

To create an optimized build for deployment on Cloudflare Pages:

```bash
npm run build
```

The generated static assets live in the `dist/` directory.

## 🧪 Quality checklist

- Responsive layout tuned for both mobile and desktop breakpoints.
- Accessibility-aware colour choices and descriptive link text.
- Strict separation of presentation and data for easy editing.
